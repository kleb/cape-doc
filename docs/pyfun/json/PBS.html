5.2.2. PBS Script Options¶

The “PBS” section of pyFun.json controls settings for PBS scripts when FUN3D runs are submitted as PBS jobs to a high-performance computing environment. Regardless of whether or not the user intends to submit PBS jobs, pyFun creates scripts called run_fun3d.pbs or run_fun.01.pbs, run_fun3d.02.pbs, etc. At the top of these files is a collection of PBS directives, which will be ignored if the script is launched locally.

To run such a script locally without submitting it, call it with the appropriate shell.

$ bash run_fun3d.pbs

If the job is to be submitted, simply submit it. A special script called pqsub is recommended over the usual qsub because it also creates a file jobID.dat that saves the PBS job number

$ pqsub run_fun3d.pbs

The full description of the PBS section can be found in the corresponding Cape PBS section.
