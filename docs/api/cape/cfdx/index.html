cape.cfdx: Templates for CFD control toolsÂ¶

This is a container module for templates to the individual CFD solver executives:

  • cape.pycart

  • cape.pyfun

  • cape.pyover

Not all of the modules templated in these specific modules are derived from cape.cfdx modules, but modules and actions that are specific to CFD generally are.

  • cape.cfdx.bin: CAPE executable operation
  • cape.cfdx.case: Generic case control
  • cape.cfdx.cmd: Creating system commands
  • cape.cfdx.dataBook: CFD Data book nodule
  • cape.cfdx.lineLoad: Sectional loads databook
  • cape.cfdx.pointSensor: Point sensor databook
  • cape.cfdx.queue: PBS and Slurm queue interface module
  • cape.cfdx.report: Automated CFD report interface
  • cape.cfdx.volcomp: Cell volume computation tools
  • cape.cfdx.options: Options module
  • cape.cfdx.cli_doc: Template help messages for CAPE executables
