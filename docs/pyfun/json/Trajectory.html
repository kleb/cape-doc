RunMatrix or Run MatrixÂ¶

A required section of pyFun.json describes the variables that define the run conditions and the values of those variables. In many cases, these will be familiar input variables like Mach number, angle of attack, and sideslip angle; but in other cases they will require additional description within the pyFun.json file. The generic run matrix description is found in the Cape RunMatrix section.

The default list of keys is mach, alpha, beta, Re, T.
