4.3.10. Generic Cape Geometry Module: cape.geom¶

This module provides several methods for modifying points or performing other geometric manipulations in a way accessible to each of the subclasses.

Versions:
  • 2015-09-30 @ddalle: First version
cape.geom.DistancePointToCurve(x, X)¶

Get distance from a point to each segment of a piecewise linear curve

Call:
>>> D = DistancePointToCurve(x, X)
Inputs:
x: np.ndarray shape=(3,)

Test point

X: np.ndarray shape=(n,3)

Array of curve break points
Outputs:
D: np.ndarray shape=(n-1,)

Distance from x to each segment
Versions:
  • 2016-09-29 @ddalle: First version
cape.geom.DistancePointToLine(x, x1, x2)¶

Get distance from a point to a line segment

Call:
>>> d = DistancePointToLine(x, x1, x2)
Inputs:
x: np.ndarray shape=(3,)

Test point

x1: np.ndarray shape=(3,)

Segment start point

x2: np.ndarray shape=(3,)

Segment end point
Outputs:
d: float

Distance from segment to point
Versions:
  • 2016-09-29 @ddalle: First version
cape.geom.RotatePoints(X, v1, v2, theta)¶

Rotate a list of points

Inputs:
X: numpy.ndarray`(:class:`float), shape = (N,3)

List of node coordinates

v1: numpy.ndarray, shape = (3,)

Start point of rotation vector

v2: numpy.ndarray, shape = (3,)

End point of rotation vector

theta: float

Rotation angle in degrees
Outputs:
Y: numpy.ndarray`(:class:`float), shape = (N,3)

List of rotated node coordinates

cape.geom.TranslatePoints(X, dR)¶

Translate the nodes of a triangulation object.

The offset coordinates may be specified as individual inputs or a single vector of three coordinates.

Call:
>>> TranslatePoints(X, dR)
Inputs:
X: numpy.ndarray`(:class:`float), shape = (N,3)

List of node coordinates

dR: numpy.ndarray or list

List of three coordinates to use for translation
Outputs:
Y: numpy.ndarray`(:class:`float), shape = (N,3)

List of translated node coordinates
Versions:
  • 2014-10-08 @ddalle: Copied from previous TriBase.Translate()
