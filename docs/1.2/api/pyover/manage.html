cape.pyover.manage: Manage pyOver case folders¶

cape.pyover.manage: Manage pyover case folders¶

This module is a derivative of the main solution folder management module cape.manage for OVERFLOW. It provides OVERFLOW-specific versions of the three top-level functions, which each correspond to a primary command-line option.

Function

Command-line

CleanFolder()

--clean

ArchiveFolder()

--archive

SkeletonFolder()

--skeleton

The --unarchive command does not require any specific customizations for OVERFLOW, and the generic version of cape.manage.UnarchiveFolder() is just called directly.

See also:

  • cape.manage

  • cape.pyover.options.archiveopts

  • cape.cfdx.options.archiveopts
