4.3.22.7. Geometry configuration options¶

This module interfaces options for geometric configurations. It defines quantities such as reference area, moment reference points, and other points to track.

class cape.options.Config.Config(fname=None, **kw)¶

Dictionary-based interfaced for surface configuration

It is primarily used for naming surface components, grouping them, defining moment reference points, defining other points, and requesting components of interest.

Call:
>>> opts = Config(**kw)
Inputs:
kw: dict

Dictionary of configuration
Outputs:
opts: cape.options.Config.Config

Surface configuration options interface
