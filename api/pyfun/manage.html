cape.pyfun.manage: Manage pyFun case folders¶

This module is a derivative of the main solution folder management module cape.manage. It provides FUN3D-specific versions of the three top-level functions, which each correspond to a primary command-line option.

Function

Command-line

CleanFolder()

--clean

ArchiveFolder()

--archive

SkeletonFolder()

--skeleton

The FUN3D-specific versions of these commands use the function pyFun.options.Archive.auto_Archive(), which apply the default settings appropriate to pyFun, and then call the generic version of the function with the same name from cape.manage. In addition, this module sets

# Subdirectories
fsub = ["Flow"]

which instructs the archiving functions to also look inside the folder Flow/ if it exists.

The --unarchive command does not require any specific customizations for FUN3D, and the generic version of cape.manage.UnarchiveFolder() is just called directly.

cape.pyfun.manage.ArchiveFolder(opts, phantom=False)¶

Archive a folder to a backup location and clean up nonessential files

Inputs

Options interface including management/archive interface
