cape.pycart.preSpecCntl: Cart3D preSpec.c3d.cntl Interface¶

This is a module built off of the cape.filecntl.FileCntl class customized for manipulating preSpec.c3d.cntl files. Such files are split into section by lines of the format

$__Prespecified_Adaptation_Regions

and this module is designed to recognize such sections, although this is the only section. The main feature of this module is to add or remove additional refinement boxes and additional XLev surface refinements.

Cart3D preSpec.c3d.cntl: pyCart.preSpecCntl.PreSpecCntl¶

class cape.pycart.preSpecCntl.PreSpecCntl(fname='preSpec.c3d.cntl')¶

File control class for preSpec.c3d.cntl files

This class is derived from the pyCart.fileCntl.FileCntl class, so all methods applicable to that class can also be used for instances of this class.

Call
>>> preSpec = pyCart.PreSpecCntl()
>>> preSpec = pyCart.PreSpecCntl(fname)
Inputs
fname: str

Name of CNTL file to read, defaults to 'preSpec.c3d.cntl'
