5.4.9. Post-processing module: pyCart.post¶

This is the pyCart post-processing module. Primarily it is focused on reading loadsCC.dat and loadsTRI.dat files.

5.4.9.1. Cart3D “loads.dat” class: pyCart.post.LoadsDat¶

class pyCart.post.LoadsDat(cart3d=None, fname='loadsCC.dat')¶

Class to store force and moment coefficients

Call:
>>> FM = pyCart.LoadsDat(cart3d=None, fname="loadsCC.dat")
Inputs:
cart3d: pyCart.cart3d.Cart3d

Overall pyCart control instance

fname: str

Name of individual files to read
Outputs:
FM: pyCart.post.LoadsDat

Instance of this class
Data members:
Aref: float

Reference area

Lref: float

Reference length

MRP: numpy.ndarray, shape=(3,)

Moment reference point

fname: str

Name of individual files to read

Components: list (str)

List of component names found

C_A: dict

Dictionary of axial force coefficient for each component
NewComponent(compID, nCase)¶

Create a new component with coefficient vector of correct size.

Call:
>>> FM.NewComponent(compID, nCase)
Inputs:
FM: pyCart.post.LoadsDat

Instance of force-and-moment object

compID: str

Name of component to add

nCase: int

Number of cases in trajectoy (needed to initialize new comps)
Outputs:

None

This function creates a vector of NaN if comp is not in the list of existing components.

ReadLoadsFile(fname, i, nCase)¶

Read a “loads*.dat” file and process each line

Call:
>>> FM.ReadLoadsFile(fname, i, nCase)
Inputs:
FM: pyCart.post.LoadsDat

Instance of force-and-moment object

fname: str

Path to file to read

i: int

Case number

nCase: int

Number of cases in trajectoy (needed to initialize new comps)
Outputs:

None

This function updates the FM.C_A, FM.C_Y, etc. fields and adds components to FM.Components if necessary.

Write(T, compID=None)¶

Write loads data to files.

A comma-separated values file is generated. An example name of the file is “loadsCC.csv”.

Call:
>>> FM.Write(T, compID=None)
Inputs:
T: pyCart.trajectory.Trajectory

Trajectory matching the conditions saved

compID: list

List of components to process, all are written by default
