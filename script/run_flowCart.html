7.4. Run Cart3D for one index of one case: run_flowCart.py¶

This script determines the appropriate index to run for an individual case (e.g. if a restart is appropriate, etc.), sets that case up, and runs it.

Call
$ run_flowCart.py [OPTIONS]
Options
-h, --help

Display this help message and quit

--verify

Run verify if starting from iteration 0

--intersect

Run intersect if starting from iteration 0

Versions

  • 2014-10-02 @ddalle: First version

  • 2015-02-14 @ddalle: Added verify and intersect checks

Methods:

This script can also be imported as a module if the scriptlib/ folder is on the Python path. If so, the following methods are created.

run_flowCart.run_flowCart(verify=False, isect=False)¶

Calls pyCart.case.run_flowCart()

Call
>>> run_flowCart(verify=False, isect=False)
Inputs
verify: bool

Whether or not to run verify before running flowCart

isect: bool

Whether or not to run intersect before running flowCart

Versions

  • 2014-10-02 @ddalle: First version

  • 2015-02-14 @ddalle: Added verify and intersect checks
