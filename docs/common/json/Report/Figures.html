3.7.8.3. Figure DefinitionsÂ¶

Each figure contains a small number of options used to define the figure. The primary option is a list of subfigures, and the others are also defined below.

Figures: {{}} | {fig: F} | dict (dict)

Dictionary of figure definitions

fig: str

Name of figure

F: dict

Dictionary of settings for figure called "F"

Header: str

Title to be placed at the top of the figure

Alignment: {"left"} | "center" | "right"

Horizontal alignment for the figure

Subfigures: [] | list (str)

List of subfigures
