4.3.22.8. Data book options module¶

This module contains the basic interface for data book options generic to all solvers. Some options are not generic, and so the derivative options classes such as cape.options.DataBook.DataBook have additional methods.

class cape.options.DataBook.DataBook(fname=None, **kw)¶

Dictionary-based interface for DataBook specifications

Call:

Inputs:

Dictionary of options
