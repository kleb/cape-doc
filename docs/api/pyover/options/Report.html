cape.pyover.options.Report: pyOver Report Options¶

This module provides options for creating automated reports for pyOver results. Although there are some subfigure types that are unique to pyOver, all of the methods in the pyOver.options.Report.Report class are inherited from cape version.

See the JSON Report section . for the list of available figures and subfigures (along with other options).
