cape.pyfun.options.DataBook: Data book options¶

This module contains the interface for data book options for pyFun and FUN3D. The classes in this module are subclassed as

  • pyFun.options.DataBook.DataBook -> cape.cfdx.options.DataBook.DataBook

  • pyFun.options.DataBook.DBTarget -> cape.cfdx.options.DataBook.DBTarget

The FUN3D-specific options for these classes are almost null, but a few methods are modified in order to change default data book component types and the columns of data available for each.
