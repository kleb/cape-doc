4.3.14. Point sensors module: cape.pointSensor¶

This module contains a class for reading and averaging point sensors. It is not included in the cape.dataBook module in order to give finer import control when used in other modules.

Point sensors are set into groups, so the "DataBook" section of the JSON file may have a point sensor group called "P1" that includes points "p1", "p2", and "p3".

If a data book is read in as DB, the point sensor group DBP for group "P1" and the point sensor p1 are obtained using the commands below.

// Point sensor group
DBP = DB.PointSensors["P1"]
// Individual point sensor
p1 = DBP["p1"]
